#  WMS Master Admin - Warehouse Management System

Template dashboard admin modern untuk manajemen gudang yang dibangun dengan **React 19**, **Tailwind CSS**, dan **TypeScript**. Aplikasi ini dirancang untuk kecepatan, kemudahan penggunaan, dan estetika profesional.

---

##  Fitur Utama

-  Dashboard Analytics**: Pantau statistik inventaris secara real-time (Total Barang, Nilai Inventaris, Stok Menipis, dan Stok Habis).
-  Full CRUD Operations**: Tambah, lihat, ubah, dan hapus data inventaris dengan antarmuka modal yang intuitif.
-  Smart Search & Filtering**: Temukan barang berdasarkan nama atau SKU, serta filter berdasarkan kategori.
-  Responsive & Collapsible Sidebar**: Navigasi yang fleksibel untuk kenyamanan maksimal di berbagai ukuran layar.
- LocalStorage Persistence**: Data Anda tetap tersimpan di browser meskipun halaman di-refresh.
- Auto-Status System**: Penentuan status barang otomatis (In Stock, Low Stock, Out of Stock) berdasarkan kuantitas.

---

## Petunjuk Penggunaan

### 1. Menambah Barang Baru
Klik tombol **"Add New Item"** di pojok kanan atas dashboard. Isi formulir lengkap (Nama, SKU, Kategori, Kuantitas, Harga, dan Lokasi), lalu klik "Add Item".

### 2. Mengedit Data
Arahkan kursor ke baris barang di tabel inventaris. Klik ikon **Pensil (Edit)** yang muncul di kolom paling kanan. Ubah data yang diinginkan dan klik "Save Changes".

### 3. Menghapus Barang
Klik ikon **Tempat Sampah (Delete)** pada baris barang yang ingin dihapus. Konfirmasi tindakan pada pop-up browser.

### 4. Pencarian dan Filter
Gunakan kotak pencarian untuk mencari berdasarkan Nama atau SKU. Gunakan menu dropdown kategori untuk mempersempit daftar tampilan barang.

### 5. Navigasi Sidebar
Gunakan ikon **Hamburger** di atas logo untuk menciutkan (*collapse*) sidebar guna mendapatkan ruang kerja yang lebih luas.

## Teknologi yang Digunakan

- **Framework**: React 19
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide-inspired SVG System
- **Data Fetching**: ESM Modules via esm.sh
